# Tag registry

Single canonical list for the vault. All tags used in notes must appear here.

## Rules

- Tags must be kebab-case, lowercase.
- Max 5 tags per note.
- Before using a tag, confirm it exists here. If it doesn't, add it to the Draft section at the bottom first, then use it.
- `validate.py` warns on tags not in this list.
- Prefer broad tags over hyper-specific ones. Tag the domain, not the instance (`immunotherapy` not `btk-inhibitors`).

## Atom sub-types

Use alongside a primary domain tag to distinguish atom varieties.

- `cognitive-bias`, systematic deviation from rational judgment
- `concept`, named idea that organises a domain
- `event`, historically significant occurrence
- `framework`, structured model for thinking about a domain
- `heuristic`, practical rule of thumb from experience
- `mechanism`, causal process explaining how something works
- `person`, historically significant individual
- `school-of-thought`, named intellectual movement or tradition
- `theorem`, formally provable or empirically established result
- `tool`, method, technique, or instrument applied to problems
